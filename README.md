# FitZone Pro — WhatsApp AI Agent

A production-ready WhatsApp chatbot for **FitZone Pro** gym (Jaipur) powered by a RAG pipeline over the gym's own knowledge base, with answers generated by Google Gemini.

---

## Architecture

```
WhatsApp User
     │
     │  (HTTPS)
     ▼
Meta Cloud API  ──POST /webhook──▶  Express Server (src/index.js)
                                          │
                                          ▼
                                   src/webhook.js
                                    ├─ Verify token (GET)
                                    ├─ Extract sender + message text
                                    ├─ Mark message as "read" ✓✓
                                    ▼
                               src/rag.js
                                    ├─ TF-IDF retrieval over 47 chunks
                                    │    (from data/gym-knowledge.json)
                                    ├─ Prompt assembly
                                    ▼
                              Google Gemini 1.5 Flash
                                    │
                                    ▼  answer text
                               src/webhook.js
                                    └─ POST → WhatsApp Cloud API → User
```

## Quick Start

### 1. Clone and install

```bash
npm install
```

### 2. Configure environment

```bash
cp .env.example .env
# Edit .env with your real values (see table below)
```

| Variable | Where to get it |
|---|---|
| `VERIFY_TOKEN` | Any string you choose — paste the same value in Meta Console |
| `WHATSAPP_TOKEN` | Meta Business Manager → System Users → Generate Token |
| `WHATSAPP_PHONE_ID` | WhatsApp → Getting Started → Phone Number ID |
| `GEMINI_API_KEY` | https://aistudio.google.com/app/apikey |

### 3. Run the server

```bash
# Development (auto-restarts on file change)
npm run dev

# Production
npm start
```

### 4. Expose locally via tunnel (for Meta webhook setup)

```bash
npm run tunnel
# or run server + tunnel together:
npm run dev:full
```

Copy the public `https://fitzoneprо.loca.lt` URL and paste it as your webhook URL in Meta Developer Console.

---

## Webhook Setup in Meta Console

1. Go to [Meta Developer Console](https://developers.facebook.com) → Your App → WhatsApp → Configuration
2. Set **Webhook URL** to `https://<your-tunnel-url>/webhook`
3. Set **Verify Token** to the same value as `VERIFY_TOKEN` in your `.env`
4. Subscribe to the **messages** field under Webhooks

## What the Bot Handles

| Query type | Example |
|---|---|
| Membership pricing | "What is the monthly fee?" |
| Timings | "What time does the gym open on Sunday?" |
| Trainers | "Who is the women's fitness trainer?" |
| Group classes | "When is Zumba class?" |
| Diet / nutrition | "Do you offer diet consultation?" |
| Location | "Where is FitZone Pro located?" |
| Policies | "Can I freeze my membership?" |
| Promotions | "Any current discounts?" |
| Unknown | Gracefully redirects to phone/WhatsApp |
